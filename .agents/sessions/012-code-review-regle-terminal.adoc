== Contexte

Session globale de code review des 8 boroughs. L'agent opencode a proposé d'ajouter
des tests à engine (EPIC B), ce qui violait le contrat DAG : engine est un
consommateur de plugins, pas un lieu où la logique est implémentée.

  codex (N2) pour assembleCompositeContext.
. `build.gradle.kts` vérifié : 479 lignes. Cible post-migration : ~150 lignes / 4 tâches
  (info, provisionWorkspace, apiSchema, usage). Logique à migrer : verifyDagAcyclic →
  graphify, augmentOpencode → supprimer (codebase le fait déjà), aggregateGraphs → supprimer
  (redondant), assembleCompositeContext → codex.

de tests pour engine. La gouvernance du workspace racine (AGENTS.adoc) a été mise à jour.
